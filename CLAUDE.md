# My System

Персональная система управления жизнью и продуктивностью.

## Философия

Система построена на трёх методологиях:

### 1. GTD (Getting Things Done)
- Inbox для быстрого сбора
- Someday/Maybe для идей
- Weekly Review для синхронизации
- Контексты (энергия, время)

### 2. @whysasha — Система бесконечной мотивации
**Ключевой принцип:** Мозг — помощник, а не враг.

- **Награда ТОЛЬКО за результат** — никакого "халявного" дофамина
- **Больше результат = больше награда** — соразмерность
- **Награда сразу** — немедленное подкрепление
- **Личный список наград** — что кайфует именно тебя
- **Анти-кортизол** — празднуем победы, не стыдим за пропуски

Мозг формирует дофаминовую привязку: `работа → результат → награда → кайф → хочу ещё`

### 3. Kaizen (改善)
- Маленькие ежедневные улучшения
- Реалистичные цели (3 задачи в день)
- Рефлексия и инсайты
- Непрерывное улучшение

## Общие принципы кода

### Messaging (для ботов, UI)
- **✅ DO:** "Отлично! +50₽ за утренний кайдзен!"
- **❌ DON'T:** "Ты пропустил утро", "Мог бы лучше"
- **Правило:** Празднуем победы, не стыдим за провалы

### Цели и задачи
- **Реалистичные:** Лучше выполнить 3 задачи, чем провалить 10
- **Измеримые:** "Заработать 50₽" вместо "Поработать"
- **Анти-кортизол:** Невыполнение ≠ стыд, просто нейтральность

### Code Style
- Type hints везде
- Docstrings для неочевидных функций
- TODO комментарии для известных ограничений
- Async/await где возможно
